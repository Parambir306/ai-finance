🧳 AI Finance Management — Smart Money Tracking Made Easy!

AI Finance Management is a smart web app built using React.js (for UI), Node.js & Express.js (for backend), and MongoDB (for data storage). It helps users track income, expenses, and savings with AI-powered categorization, receipt scanning, and real-time analytics.

🧠 Overview

The application simplifies personal finance management by automating expense tracking, categorizing transactions using AI, and providing meaningful insights for better budgeting decisions.

It also includes a smart receipt scanner to extract data directly from images.

🛠️ Built With

React.js – Frontend UI

Node.js – Backend runtime

Express.js – Server framework

MongoDB – Database

Python – AI logic

TensorFlow/Keras – ML models

OpenCV – Receipt scanning

OpenAI API – Generative AI


✨ Key Features

📊 Real-time income & expense tracking

🤖 AI-based transaction categorization

🧾 Smart Receipt Scanner (extracts data from images)

📈 Budget insights & analytics

⚡ Automated expense logging


⚙️ How It Works

User inputs transaction or uploads a receipt

Image is processed using OpenCV

AI model extracts and categorizes the data

Data is stored in MongoDB

Dashboard displays analytics and insights
